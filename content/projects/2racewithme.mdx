## Overview

Developed at the Interactive Media Lab, 2RaceWithMe is an interactive cycling system that combines physical hardware with immersive software to encourage exercise among older adults.

## Key Features

- **Arduino sensor integration** for real-time cadence and resistance tracking
- **Unity 3D environment** that responds to cycling input with immersive visuals
- **React dashboard** for tracking progress and setting goals
- **Adaptive difficulty** that adjusts to individual fitness levels
