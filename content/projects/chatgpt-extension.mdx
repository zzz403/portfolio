## Overview

A Chrome extension with 500+ downloads that adds power-user features to ChatGPT.

## Key Features

- **Conversation export** to Markdown, PDF, and plain text
- **Custom prompt library** with quick-insert shortcuts
- **Improved formatting** with syntax highlighting and better code blocks
- **Keyboard shortcuts** for faster navigation and interaction
