## Overview

YouWo AI is a cross-platform AI-powered note-taking and learning assistant. Import anything — lectures, PDFs, YouTube videos — and chat with an AI that remembers everything and answers with source citations.

## Key Features

- **Real-time transcription** with 98% accuracy in 50+ languages
- **PDF & document intelligence** with automatic chapter splitting
- **YouTube video analysis** with clickable timestamped transcripts
- **AI chat with citations** — every answer links back to exact sources
- **One-click study tools** — generate flashcards, quizzes, and summaries

## Tech Stack

- React + TypeScript frontend
- Multi-model AI backend (GPT-4, Claude, Gemini)
- Cross-platform: iOS, Android, Web
