## Overview

Built at Siemens Healthineers, this project processes hearing test data collected from remote areas to enable personalized hearing aid fitting and adaptation.

## Key Contributions

- **Data pipeline architecture** for ingesting and normalizing heterogeneous hearing test formats
- **Statistical analysis** of audiometric data to identify patterns across demographics
- **Personalization engine** that maps hearing profiles to device configuration parameters
- **Quality assurance** framework ensuring data integrity across the pipeline
