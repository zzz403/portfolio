## Overview

A Ren'Py-based visual novel built in partnership with the Toronto forensic police department, using interactive storytelling to teach real-world forensic investigation methods. The game was developed with direct input from forensic professionals to ensure accuracy and educational value.

## Key Features

- **Authentic forensic education** — content developed with Toronto's forensic police unit, reflecting real investigative procedures
- **Institutional collaboration** — worked directly with forensic department advisors to validate case scenarios and techniques
- **Interactive investigation** — players collect evidence, analyze clues, and apply forensic reasoning to solve cases
- **Ren'Py engine** — built with Python-based visual novel framework, supporting branching narrative and rich media
