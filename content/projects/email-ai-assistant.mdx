## Overview

An AI-powered email management system achieving 94% accuracy in categorization and response drafting.

## Key Features

- **Smart categorization** — automatically sorts emails by urgency, topic, and action required
- **Priority scoring** using ML models trained on user behavior patterns
- **Draft generation** — context-aware response suggestions with one-click send
- **Learning loop** — improves over time based on user corrections and preferences
